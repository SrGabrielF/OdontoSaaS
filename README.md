# OdontoSaaS 🦷

Sistema SaaS para gestão de clínicas odontológicas.

## Tecnologias

- React
- Vite
- JavaScript
- TailwindCSS

## Funcionalidades

- Cadastro de pacientes
- Agenda de consultas
- Diagnóstico odontológico
- Controle de tratamentos
- Sistema de pagamentos
- Dashboard administrativo

## Rodar o projeto

Pré-requisitos:
Node.js instalado

1. Instalar dependências

npm install

2. Rodar o projeto

npm run dev
